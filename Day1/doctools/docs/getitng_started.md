# Getting Started
Welcome to the documentation! This page will guide you through setting up your environment and running the project locally.

# Install Dependencies
```bash
pip install mkdocs-material
```

# Project Structure
```
my-docs/
├── docs/
│   ├── index.md
│   ├── getting-started.md
│   └── diagrams.md
├── mkdocs.yml
```

# Start the Docs Site
```bash
mkdocs server
```
